## CURSO DE PROGRAMACION EN PYTHON - UNSAM

Como lo definieron los profesores

*"...El objetivo de este curso es enseñar los fundamentos del lenguaje Python orientado al manejo de datos, a la escritura de scripts y a una organización adecuada de los programas.*

*Estudiaremos un poco de teoría de algoritmos, incluyendo conceptos básicos de la teoría de la complejidad y algunas estructuras de datos no triviales. Introduciremos la programación orientada a objetos.*

*Al final de este curso deberían poder escribir sus propios programas en Python así como leer y modificar programas escritos por otros."*

___

### CONTENIDOS

- Introduccion a Python
- Estructuras y funciones
- Trabajando con datos
- Comprensión de listas
- Aleatoridad
- Complejidad y Organización de programas
- Diseño, especificación, documentación y estilo.
- Pandas
- Clases y objetos
- Generadores e iteradores
- Recursión y regresión lineal
- Ordenamiento (sort / supervised classification)
